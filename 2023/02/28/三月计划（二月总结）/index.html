三月计划

part1. 二月总结

done：

准备考研的第一个月，查看相关院校，进行考研知识的扫盲，确定出了几所分阶梯难度的预备院校，并对大概的考研常识进行了了解。

英语学习方面：因为英语重点在于积累，做到了每日单词打卡，每日长难句学习等，记录下载自己学习的足迹，另外还在准备3月的CET6，还没有来得及谢谢考研英语的阅读。

数学学习方面：数学购买资料，查找网课，目前网课一轮复习进度比较慢，等待3月加快一点速度，多给数学分一点时间，目前的数学学习时间偏少。

技术方面：还在准备外包大赛的项目，项目计划也比较落后，主要因为小组成员都才返校，各有各的工作。我主要在学习ES和pytorch相关的技术，但是进度也才入门（主要为了答辩花费很多时间）。

专业课方面：还没开始

改进：

  1. 院校的具体招生简章需要查看，分析自己的学习路线
  2. 英语继续坚持每日打卡等，6级前保证3-4小时的英语学习，6级后英语学习减少到2小时左右
  3. 数学学习加快一点速度
  4. 专业课开始准备
  5. 项目进度方面，准备工作基本已经完成，开始实际操作

part3. 三月计划

todo:

  • 继续查看相关院校信息，广撒网进行查看
  • 查看考研黑名单并罗列在研究生博客中
  • 对于有意向的院校进行查看，查看招生简章，推荐资料等
  • 学习时间安排：days：周六下午—周五下午 休息：周五晚上—周六上午
                         time：早上大概8.30到晚上8.30或者9.30（中午12点—下午两点，两个小时指标）

数学学习：

主要是抓进度的同时抓质量，质量怎么抓？写题吧（太简单粗暴）

  • 时间上：提高数学学习时间，

    1. 六级前：数学时间应该在3小时左右，分配在下午和晚上，下午4.30到5.30左右，晚上两个小时
    2. 六级后：减少英语学习时间，增加数学时间，包含数学时间5个小时 上午（2个小时）：分两个小时给数学，英语一个小时（9，10，11）。下午不变（1），晚上不变（2）

  • 进度上：完成前三章的学习任务，660开始练习，错题整理进度跟上

  • 知识上：高等数学的课本，或者说该学校的高数参考数加入复习清单

英语学习：

英语重点还是在词汇语法和阅读上

  • 首先是时间上的问题，
    1. 六级前复习英语时间保持在3-4小时，上午大概都会给到英语大概三个小时，晚上还有一部分时间
    2. 六级后复习英语时间在1-2小时，上午（1），晚上（1）
  • 进度问题
    1. 每日单词打卡
    2. 每日长难句学习
    3. 三月完成六级考试，10篇的阅读训练（3天1篇）
    4. 语法等的复建

专业课学习：

408跑不掉咯

  • 预计在六级考试后开始
  • 书籍购买：六级考试前进行
  • 网课资源准备：六级考试前进行
  • 时间：下午专业课复习（1-2个小时）

项目学习：

  • 主要是会议，技术准备和代码，这些时间都不定，变动性很大
  • 时间设置为下午4.30前，其他任务耽搁的进度可以在此挽回
  • 提高项目进度，4月过要交作品，需要构建出可运行大体

春招：

  • 这一模块随机性更大了，只能说随缘，进入三月再考虑吧
